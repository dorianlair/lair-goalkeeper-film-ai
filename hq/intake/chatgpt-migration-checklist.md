# ChatGPT → HQ migration checklist

## 1) Export

- [ ] Export ChatGPT data from OpenAI settings.
- [ ] Save zip/tar exactly as downloaded.
- [ ] Put it under `hq/archive/` with date in filename.

## 2) Sanitize

- [ ] Remove passwords, API keys, tokens, private IDs.
- [ ] Remove sensitive third-party personal data you do not need.
- [ ] Keep a private local-only copy if needed.

## 3) Distill into HQ

- [ ] Preferences → `hq/preferences.md`
- [ ] Business decisions/context → `hq/business.md` + `hq/decisions.md`
- [ ] Life systems/routines → `hq/life.md`
- [ ] Career strategy → `hq/career.md`
- [ ] Learnings/notes/frameworks → `hq/knowledge.md`
- [ ] Outcomes/targets → `hq/goals.md`
- [ ] Long-horizon intent/legacy → `hq/legacy.md`

## 4) Activate cadence

- [ ] Update `hq/dashboard.md`
- [ ] Update `hq/next-actions.md`
- [ ] Run weekly review in `hq/weekly-review.md`
