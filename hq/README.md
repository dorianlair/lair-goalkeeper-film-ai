# HQ

This folder is your personal operating system.

## How to use

1. Put raw exports in `hq/archive/`.
2. Copy key insights into domain files (`business.md`, `life.md`, etc.).
3. Keep `dashboard.md` and `next-actions.md` updated weekly.
4. Record major choices in `decisions.md`.

## Source-of-truth rule

- Raw data is preserved in `archive/`.
- Structured truth lives in the domain files.
- If there is a conflict, update structured files and note the decision.
