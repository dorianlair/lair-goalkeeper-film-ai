Game Film Review

A local coach dashboard for uploading goalkeeper footage, prompting Gemini, and turning match clips into usable coaching feedback.

How it works

  1. Upload a game clip
  2. Add context like opponent, goals, and notes
  3. Run analysis
  4. Review coaching feedback and next steps

Best for

  • Goalkeeper post-game reviews
  • Tryout / showcase feedback
  • Coach-player development sessions
